## Git Pull
 Fetches the newest changes from GitHub and immediately merges them into your local files.
## git clone
 Downloads a complete copy of a remote GitHub repository to your local computer for the first time.
## git add
Stages all of your modified and new local files, preparing them to be saved.
## git commit
aves your staged changes locally with a descriptive note.
